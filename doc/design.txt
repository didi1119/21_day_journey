## 🎯 專案總覽

### 願景聲明
透過結合心理學、自然療癒與數位科技，創造一個引導現代人重新連結內在、他人與自然的沉浸式體驗平台。

### 核心價值
- **自我覺察** - 透過心理測驗深入了解自己
- **持續成長** - 21天漸進式改變計畫
- **社群連結** - 建立有意義的人際關係
- **自然智慧** - 從森林生態學習生活哲學

### 目標用戶
- **主要族群**: 25-45歲都市白領
- **痛點**: 壓力大、迷失方向、缺乏連結
- **需求**: 自我探索、心靈成長、社群歸屬

## 💻 技術規格

### 前端技術棧
```yaml
框架與函式庫:
  - HTML5: 語義化標籤結構

  - JavaScript: ES6+ 原生實現
```

### 三幕劇結構

**主題**: 走入林間小徑，與內在相遇

| 日期 | 標題 | 核心活動 | 預期成果 |
|-----|------|---------|---------|
| Day 1 | 歡迎來到森林 | 設定意圖儀式 | 建立承諾 |
| Day 2 | 傾聽身體智慧 | 身體掃描練習 | 覺察提升 |
| Day 3 | 情緒的顏色 | 情緒日記 | 情緒識別 |
| Day 4 | 內在小孩 | 童年回憶探索 | 自我理解 |
| Day 5 | 陰影工作 | 接納練習 | 整合陰影 |
| Day 6 | 價值羅盤 | 價值觀釐清 | 方向確立 |
| Day 7 | 第一週整合 | 回顧與慶祝 | 階段總結 |

#### 第二幕：深入寧靜森林 (Day 8-14)
**主題**: 在森林深處，與他人連結

| 日期 | 標題 | 核心活動 | 預期成果 |
|-----|------|---------|---------|
| Day 8 | 關係的鏡子 | 關係地圖繪製 | 關係覺察 |
| Day 9 | 同理心練習 | 換位思考 | 理解他人 |
| Day 10 | 界線的藝術 | 設立健康界線 | 自我保護 |
| Day 11 | 衝突的禮物 | 衝突轉化 | 和諧相處 |
| Day 12 | 感恩的力量 | 感恩信練習 | 關係深化 |
| Day 13 | 給予與接受 | 能量交換 | 平衡關係 |
| Day 14 | 第二週整合 | 關係回顧 | 階段總結 |

#### 第三幕：帶一片森林回家 (Day 15-21)
**主題**: 將森林智慧，編織進生活

| 日期 | 標題 | 核心活動 | 預期成果 |

## ⚡ 效能優化

### 載入優化
```javascript
// 關鍵指標目標
{
  FCP: < 1.8s,   // First Contentful Paint
  LCP: < 2.5s,   // Largest Contentful Paint
  FID: < 100ms,  // First Input Delay
  CLS: < 0.1     // Cumulative Layout Shift
}
```

### 優化策略

#### 1. 資源優化
- 圖片：WebP格式 + 延遲載入
- 字型：預載入 + font-display: swap
- CSS：Critical CSS內聯
- JS：程式碼分割 + 異步載入

#### 2. 快取策略
```javascript
// Service Worker 快取
const cacheStrategy = {
  static: {
    // HTML, CSS, JS
    strategy: 'Cache First',
    maxAge: 7 * 24 * 60 * 60 // 7天
  },
  images: {
    strategy: 'Cache First',
    maxAge: 30 * 24 * 60 * 60 // 30天
  },
  api: {
    strategy: 'Network First',
    timeout: 3000
  }
}
```

#### 3. 程式碼優化
- 移除未使用的 CSS
- Tree shaking
- Minification
- Gzip 壓縮
